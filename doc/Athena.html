learn(traces, merge_selector \\ fn xA -> Athena.KTails.selector(2, xA) end, threshold \\ 2.0)

Learn an EFSM from a set of traces. Uses the supplied merge_selector function, which should accept an EFSM and return a list of pairs containing a merge ‘score’ and the pair of states to merge. The learner will attempt to merge the highest scoring pair, unless this produces non-determinism, in which case it will try the next pair. The learning will continue until the score of the best possible merge falls below the threshold
