Specs:

  • subsumes?(t, t) :: boolean

Checks whether the first label ‘subsumes’ the second - that is, whether any event that matches the first is certain to match the second.

This requires that the event names match, and that all of the guard expressions in the second are subsumed by at least one in the first. It also requires that the updates and outputs are identical (if sorted), since non-identical outputs or updates would produce observable differences, so the transitions can’t be considered equivilent.
